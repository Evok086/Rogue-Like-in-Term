	Tant que input(!=X):
		c = carte[x,y]
		Input = saisie_bloquante () //on attend une réponse 
		V = False
		if (input == z):
			V = Verif(personnages) <- carte[x+1],[y]
			if (v == True): 
				Personnage.position.x += 1
				fin if
		else if (input == q):
			V= Verif(personnages)<- carte[x,y+1]
			if (V == True):
				personnage.position.y = y-1
				fin if
		else if (input == d):
			V= Verif(personnages)<- carte[x,y+1]
			if (V == True):
				personnage.position.y = y+1
				fin if
		else if (input == s):
			V= Verif(personnages)<- carte[x-1, y]
			if (V == True):
				personnage.position.y = y-1
				fin if
		fin if 
	fin tant que 
fin 

Fonction Verif (perosnnage m) carte
	c= carte.[x,y]
	if (c[x+position(x), y+position(y)] == # : //si mur ou si le perso est en dehors de la carte alors
		return false
	if (c [x+ position(x), y+ position(y)] == M : //si le personnage rencontre un monstre
		combat (personnages, monstre) 
		return true
	if (c [x+ position(x), y+ position(y)] == E : //si le personnage tombe sur une case equipement
		prendre objet(personnage)
		return true

On peut aller en haut, en bas, à gauche et à droite. Quand on clique sur une touche on lance une fonction qui vérifie, dans un premier temps, si la position du joueur permet de faire le mouvement que l'utilisateur désire.
